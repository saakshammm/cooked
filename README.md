# 🚀 Run Locally

## 1. Install dependencies

```bash
pip install -r requirements.txt
```

## 2. Start the server

```bash
python -m uvicorn main:app --host 0.0.0.0 --port 8000
```

## 3. Open in browser

```bash
open http://localhost:8000
```
